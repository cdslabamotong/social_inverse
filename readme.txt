Source code and dataset associated with Social-Inverse paper submitted to NeurIPS 2022.


DC.py and DE.py contain classes related to tasks of DE and DC.

StoGraph.py and USCO.py contain classes of abstract undetermined stochastic combinatorial optimization problems




"Social Inverse" contains the implementation of Social-Inverse and pre_train models.

"DSPN", "GCN", and "Other methods" contain the implementations of other methods

"data" contains realizations, diffusion models, and samples.

See readme files in the subfolders for details.
